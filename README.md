# Random Password Generator

**Автор:** Козлицкий Максим

## Описание программы

Random Password Generator — это графическое приложение для генерации случайных паролей с настраиваемыми параметрами.

### Возможности
- Настройка длины пароля (4-32 символа)
- Выбор типов символов (заглавные, строчные буквы, цифры, спецсимволы)
- Генерация случайного пароля
- Оценка надёжности пароля
- Сохранение истории сгенерированных паролей
- Копирование пароля в буфер обмена
- Экспорт истории в JSON
- Автоматическая загрузка истории при запуске

## Требования

- Python 3.7 или выше
- Tkinter (входит в стандартную установку Python)
